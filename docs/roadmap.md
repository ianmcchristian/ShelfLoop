# ShelfLoop Roadmap

## Phase 1 — Foundation MVP

- Vite/React/Tailwind project setup
- Demo apparel data model
- RFID scan simulator
- Inventory reconciliation
- Shared replenishment task queue
- Stocker phone modal
- localStorage persistence
- Logic tests

## Phase 2 — Polish and portfolio package

- Stronger dashboard visuals
- README screenshots/GIFs
- Concise PRD and business case
- Architecture diagram
- GitHub Pages deployment

## Phase 3 — RFID realism

- Adjustable scan windows
- Fixture-specific scan confidence
- Missed reads, duplicate reads, and false negatives
- Unique tag coverage metrics across repeated scans

## Phase 4 — Digital twin concept

- Model a realistic apparel subsection layout
- Use sanitized learnings from physical RFID/LiDAR characterization
- Compare easy-read and hard-read fixture types

## Phase 5 — Production-facing extensions

- Hosted backend/event log
- Real push notifications or installable PWA
- ESL/tap-to-light integration adapter
- Store operations analytics and ROI model
